# New Drug Review Report Copilot Workspace

このフォルダは、新薬の審査報告書作成を支援するための外部作業用コーパス・テンプレート置き場です。


## Project Boundary

このフォルダは新薬審査報告書作成支援専用です。

- SaMD / プログラム医療機器の制度調査、Q&A、承認品目一覧、SaMD審査報告書wikiは `/Users/yamamotoerika/Desktop/SaMD` で扱う。
- このフォルダには、SaMD用のwikiページ、SaMD承認品目マトリクス、SaMD内部Copilot packを置かない。
- 公表済み医薬品審査報告書から作る書きぶり・論点整理は、今回の新薬、特に抗凝固薬、抗血小板薬、脳血管イベント周辺の作業に限定する。
- 内部Copilotに渡す素材は、企業申請資料や未公表情報を含まない外部作業成果物に限定する。

## 目的

公表済み審査報告書から、書きぶり、論点構造、評価項目、安全性記載、審査上の結論の型を抽出し、内部環境のCopilotで企業申請資料を扱う際に再利用できる制御素材を作る。

## 前提

- 公表済み審査報告書は外部で扱える。
- 企業からの申請資料、未公表情報、内部判断はこのフォルダに置かない。
- 外部ではテンプレート、参照パターン、マトリクス、プロンプト骨子まで作る。
- 内部Copilotでは、品目固有の事実抽出、論点整理、報告書ドラフト化を行う。

## Directory Map

- `raw_public_reports/`: 公表済み審査報告書PDF。
- `extracted_text/`: PDFから抽出したテキスト。
- `matrices/`: 品目横断の比較表。
- `patterns/`: 書きぶりパターン、章別の定型表現、論点別の参照例。
- `prompt_pack/`: 内部Copilotに渡すプロンプト、制御指示、チェックリスト。
- `internal_only_placeholders/`: 内部で記入する前提の空テンプレート。実データは置かない。
- `outputs/`: 外部作業で生成した成果物。
- `logs/`: 作業ログ。

## Corpus Strategy

今回の中心品目は、脳血管イベントに対する抗凝固薬を想定する。

参照コーパスは次の3層に分ける。

1. 薬効・疾患が近い報告書
   - 抗凝固薬
   - 抗血小板薬
   - 脳梗塞、脳血管イベント、血栓塞栓症、心房細動関連

2. 安全性論点が近い報告書
   - 出血リスク
   - 高齢者
   - 腎機能、肝機能
   - 併用薬
   - 用量調整
   - 日本人データの限定性

3. 最近の新薬審査報告書
   - 薬効は必ずしも近くなくてよい。
   - 最新の章立て、表現、審査上の結論、照会事項の書き方を参照する。

## External/Internal Boundary

外部で作るもの:

- 公表済み審査報告書の比較マトリクス。
- 論点別の書きぶりパターン。
- 最近のPMDA文体・章立ての参照メモ。
- 内部Copilot用プロンプトテンプレート。
- 品目固有情報を入れるための空欄テンプレート。

内部でだけ扱うもの:

- 申請資料から抽出した品目固有情報。
- 未公表の試験データ。
- 照会事項案、審査判断案、内部検討メモ。
- 最終的な審査報告書ドラフト。
