# Reference Public Review Reports README

内部Copilot用のプロンプト、runbook、Excelマトリクス等は、別ファイル `outputs/internal_copilot_pack_2026-05-16.zip` に入っています。

## Purpose

7項の章立て、表の作り方、数値の置き方、本文の粒度を原本PDFで確認するために使います。

公表PDFは、今回品目の事実補完には使わないでください。今回品目の事実は、申請資料、CTD、試験総括報告書、照会回答等の内部資料に根拠があるものだけを使います。
